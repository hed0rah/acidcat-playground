Databending Gallery

One short synthesized arp, run through the playground's forge and sonify tools. Each card is a real, byte-exact transform: hear the before/after and see the file's bytes laid out as an image. Nothing is a plugin or a DSP effect, these are edits to the container and sample bytes that leave a still-playable file. Reproduce any of them from the command shown.

source

A 4-note arp (A / C#5 / E5 / A5), 22050 Hz mono, 16-bit PCM.

six databends

Reverse

the sample frames flipped end to end: the arp plays backwards.

python forge.py src.wav recipe wav-reverse

Bitcrush

the low byte of every 16-bit sample zeroed: crushed to 8-bit, lo-fi and gritty.

python forge.py src.wav recipe wav-bitcrush

Stutter

each short window repeated twice: a rhythmic hesitation.

python forge.py src.wav recipe wav-stutter

Data-sort

each window's bytes sorted ascending: a pixel-sort-style ramp glitch.

python forge.py src.wav recipe wav-data-sort

Sonify: invert

bytes read as an image, every pixel inverted, read back as audio.

python sonify.py bend src.wav --op invert

Sonify: rowsort

bytes as an image, each row sorted, back to audio: a smeared texture.

python sonify.py bend src.wav --op rowsort
